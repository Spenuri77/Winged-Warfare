# Winged-Warfare
Winged Warfare - gra 2D polegająca na lataniu myśliwcem i niszczeniu celów powietrznych. Celem gracza jest zdobycie jak najwyższego wyniku. Oprawa graficzna gry jest wzrowana na starych grach typu arcade.
